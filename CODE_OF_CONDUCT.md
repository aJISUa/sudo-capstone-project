# Code of Conduct

On-Care 팀과 외부 기여자가 함께 작업하는 공간에서 모두가 존중받고 생산적으로 협업할 수 있도록 본 행동 규약을 채택합니다. 본 문서는 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 의 정신을 따르되, 캡스톤 팀 환경에 맞춰 간소화한 것입니다.

---

## 1. 우리의 약속

기여자·관리자·리뷰어로서 우리는 나이·성별·신원·장애 유무·민족·경험 수준·국적·종교·기술 스택에 관계없이 모든 참여자가 **차별 없이 존중받는** 환경을 만들 것을 약속합니다.

---

## 2. 기대되는 행동

- 다른 참여자의 관점·경험에 공감하기
- 건설적인 피드백을 주고받기 (비판은 *사람이 아니라 코드·아이디어에*)
- 실수를 인정하고, 영향을 받은 이에게 사과하며, 그로부터 배우기
- 본인이 아닌 *팀과 프로젝트* 에 가장 좋은 결정 우선시
- 한국어·영어 어느 쪽으로 소통하든 친절하고 명확하게

---

## 3. 부적절한 행동

다음은 환경을 해치는 행위로 간주됩니다:

- 모욕적·비하적 발언, 인신 공격, 정치적 공격
- 성적 언어·이미지 사용, 원치 않는 성적 관심 표현
- 공개·비공개 괴롭힘
- 명시적 동의 없는 타인 개인정보(주소·이메일·실명 등) 게시
- 직업적 환경에서 부적절하다고 판단될 만한 기타 행위

---

## 4. 책임 범위

본 규약은 다음 모든 공간에 적용됩니다:

- 리포지토리 내 PR / Issue / Discussion / Wiki 댓글
- 커밋 메시지 · 코드 주석
- 프로젝트 관련 채팅(카카오톡·Discord·Slack 등)
- 프로젝트 명의의 대외 발표·데모

---

## 5. 위반 신고

규약 위반을 목격하거나 직접 겪었다면 다음 경로로 신고할 수 있습니다:

- **팀 내부**: 팀 리더(또는 신뢰하는 다른 팀원) 에게 비공개 채널로 직접 연락
- **외부 기여자**: GitHub *Report content* 기능 또는 `SECURITY.md` 의 연락처 사용

모든 신고는 비공개로 검토되며, 신고자의 신원과 진술은 보호됩니다.

---

## 6. 시행

위반이 확인될 경우 다음 중 하나 이상의 조치를 취할 수 있습니다:

1. **비공개 경고** — 위반 행위에 대한 비공개 안내와 행동 변경 요청
2. **공개 경고** — 규약 위반 사실을 댓글 등으로 공개 통보
3. **일시적 활동 제한** — 일정 기간 PR / Issue / 댓글 권한 제한
4. **영구 제한** — 반복·심각 위반 시 리포지토리 활동 영구 차단

---
